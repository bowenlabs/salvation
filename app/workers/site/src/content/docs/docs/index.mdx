Thebes is the monorepo behind two open-source Cloudflare-native projects:

- **[Cadmus](/docs/cadmus/)** — a V8-first, Cloudflare-native full-stack framework. No Node.js
  assumptions, no adapter layers, no configuration overhead.
- **[Cadmea](/docs/cadmea/)** — a free, open-source, V8-native headless CMS and admin platform,
  built on Cadmus.

Both ship as independently installable `@thebes/*` packages from npm, alongside a set of
first-party extensions — Cadmus adapters, Cadmea plugins, and a couple of standalone libraries.
See the sidebar for the full list.
